# Project: Personal Expense Tracker

## Description
A Python-based web application for managing personal expenses.
The app allows users to add, view, filter, and delete transactions. 
It also generates monthly reports and visual charts using SQLite for data storage and Matplotlib for visualization.

## Technologies Used
- Python 3
- Tkinter (GUI)
- Tkcalendar (Date picker)
- SQLite3 (Database)
- Matplotlib (Charts)

## Main Features
- Add new transactions with category and description
- Filter transactions by month and year
- View all records in a TreeView table
- Generate detailed monthly reports
- Visualize spending with pie, bar, and line charts
- Delete selected transactions

## Structure
- **ExpenseTracker**: Handles database operations
- **ExpenseTrackerGUI**: Tkinter interface
- **main()**: Entry point

## Getting Started

### Prerequisites
- Python 3.x

### Installation
1. Clone the repository:
    ```bash
    git clone https://github.com/mdadil2003/Personal-Expense-Tracker.git
    cd Personal-Expense-Tracker
    ```
2. Install required packages:
    ```bash
    pip install tk tkcalendar matplotlib
    ```
3. Run the app:
    ```bash
    python main.py
    ```
